Pywalfox

Updating these values will set the default color from the pywal palette (0-15) that should be used for certain elements of the theme.

Frame The color of the header area background

Frame inactive The color of the header area background when the browser window is inactive

Bookmark text The color of text and icons in the bookmark and find bars

Button background active The color of the background of the pressed toolbar buttons

Button background hover The color of the background of the toolbar buttons on hover

Icons The color of toolbar icons, excluding those in the find toolbar

Icons attention The color of toolbar icons in attention state such as the starred bookmark icon or finished download icon

NTP background The new tab page background color

NTP text The new tab page text color

Popup The background color of popups (such as the url bar dropdown and the arrow panels)

Popup border The border color of popups

Popup highlight The background color of items highlighted using the keyboard inside popups (such as the selected url bar dropdown item)

Popup highlight text The text color of items highlighted inside popups

Popup text The text color of popups

Sidebar The background color of the sidebar

Sidebar border The border and splitter color of the browser sidebar

Sidebar highlight The background color of highlighted rows in built-in sidebars

Sidebar highlight text The text color of highlighted rows in sidebars

Sidebar text The text color of sidebars

Tab background separator The color of the vertical separator of the background tabs

Tab background text The color of the text displayed in the inactive page tabs

Tab line The color of the selected tab line

Tab loading The color of the tab loading indicator and the tab loading burst

Tab selected The background color of the selected tab

Tab text The text color of the selected tab

Toolbar The background color for the navigation bar, the bookmarks bar, and the selected tab

Toolbar bottom separator The color of the line separating the bottom of the toolbar from the region below

Toolbar field The background color for fields in the toolbar, such as the URL bar

Toolbar field border The border color for fields in the toolbar

Toolbar field border focus The focused border color for fields in the toolbar

Toolbar field highlight The background color used to indicate the current selection of text in the URL bar

Toolbar field highlight text The color used to draw text that's currently selected in the URL bar

Toolbar field separator The color of separators inside the URL bar

Toolbar field text The color of text in fields in the toolbar, such as the URL bar

Toolbar text The color of toolbar text

Toolbar top separator The color of the line separating the top of the toolbar from the region above

Toolbar vertical separator The color of the separator next to the application menu icon.

Pywalfoxv2.0.0

Daemon update required

Update information

The currently installed version of the browser extension does not support the installed native application.
Not updating will most likely lead to unexpected behaviour and bugs.

If you encounter issues with the new version of daemon, refer to the open issues on GitHub (or create a new one).

GitHub issues

If you are migrating from extension version 1.5.3, you need to remove the old daemon located in the "Pywalfox" repo. You can safely remove the entire folder.

To install the new daemon, refer to the installation guide on the GitHub page: Installation guide

If you installed it using pip you can run the following command:

pip install --upgrade pywalfox

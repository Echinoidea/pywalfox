Update information

The native-application has received changes that are not compatible with older versions of the extension.
Not updating will most likely lead to unexpected behaviour and bugs.

If you encounter issues with the new version of daemon, refer to the open issues on GitHub (or create a new one).

GitHub issues

Depending on how you installed the daemon, you have two different options:

pip install --upgrade pywalfox cd <path to pywalfox-native repo>
git pull

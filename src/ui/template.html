Pywalfox

Fetch pywal colors to see the palette

Updating these values will set the default color from the pywal palette (0-17) that should be used for certain elements of the theme.

Colors 16 (white) and 17 (same as color0 but brighter) are special colors generated by Pywalfox and are not included in the pywal palette by default.

These values will take effect the next time you fetch pywal colors.

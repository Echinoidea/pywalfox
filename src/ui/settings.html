Palette template

Creating a custom palette

The palette is a collection of pywal colors that is generated when clicking the "Fetch pywal colors" button. A custom palette template allows you to manually select which pywal color should be the default color used for the background, text, etc.

If you want to change the color for a certain element of the browser theme, you can do that in the "Theme template"-section below.

Palette template guide

Theme template

Creating a custom theme template

Custom theme templates allows you to customize each element individually by telling Pywalfox which palette color should be used for a specific element.

If you want to use a different color than the ones provided (background, accent primary, etc), you can add another color in the "Palette template" section above.
